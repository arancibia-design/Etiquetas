Etiquetas HTML

TABLAS ETIQUETAS DE HTML

Estructuras Básicas

Etiqueta Definición Ejemplo
<html> Define el inicio y fin del documento HTML. <html> ... </html>
<head> Contiene metadatos, enlaces a estilos o scripts. <head><title>Mi sitio web</title></head>
<title> Define el título de la página web. <title>Mi sitio web</title>
<body> Contiene el contenido visible de la página. <body><p>Hola Mundo</p></body>
<!DOCTYPE html> Indica que el documento es HTML5. <!DOCTYPE html>

Texto y Formato

Etiqueta Definición Ejemplo
<h1> a <h6> Títulos de distintos niveles o tamaños. <h1>Título</h1>
<p> Párrafos de texto. <p>Hola mundo</p>
<br> Salto de línea. <p>Primera línea<br>Segunda línea</p>
<b> Texto en negritas (no semánticas). <b>Hola Mundo</b>

Estructuras del Contenido

Etiqueta Definición Ejemplo
<header> Encabezado de una página o sección. <header><h1>Mi web</h1></header>
<footer> Pie de página. <footer>©2025 Mi sitio</footer>
<main> Contenido principal. <main><h1>Contenido</h1></main>
<nav> Sección de navegación. <nav><a href="#">Inicio</a></nav>

Enlaces y Navegaciones

Etiqueta Definición Ejemplo
<a> Crea un enlace. <a href="https://feis.com">¡Visítanos!</a>
<link> Enlaza un archivo externo (como un CSS). <link rel="stylesheet" href="style.css">

Listas y Tablas

Etiqueta Definición Ejemplo
<ul> Lista sin orden (con viñetas). <ul><li>Ejemplo</li></ul>
<li> Elemento de una lista. <li>Ejemplo</li>
